What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a web page as a hierarchical tree of objects (nodes), where each node corresponds to a part of the document (like elements, attributes, or text). This tree structure allows developers to access, modify, or delete elements dynamically using programming languages such as JavaScript.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events for multiple child elements, rather than adding individual event listeners to each child element.
